## ğŸ¨ Customization

### Template Customizer
The website includes a built-in template customization tool that allows:

1. **Color Customization**
   - Primary color picker with preset options
   - Real-time preview across all template elements
   - Industry-standard color combinations

2. **Branding Options**
   - Company name customization
   - Logo upload and preview
   - Brand color synchronization

3. **Export Features**
   - Save customization settings as JSON
   - Configuration sharing with clients
   - Quick template deployment

### CSS Variables
Colors and branding are controlled through CSS variables:

```css
:root {
  --bakara-primary: #B71231;
  --bakara-secondary: #B3EC11;
  --action: #B71231;
  /* ... other variables */
}
```
